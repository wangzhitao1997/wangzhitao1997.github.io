R系linux系统上hexo博客的搭建

Created2020-11-20|Updated2020-11-20
|Post View:

1.安装git

2.安装node.js

hexo是基于node.js环境的静态博客
附：下面路径下的$path指的是你软件安装的具体

  (1)在官网下载node.js，or命令行下载：

wget http://nodejs.org/dist/v10.9.0/node-v10.9.0-linux-x64.tar.xz

 (2)解压tar -xvf xxx.tar.xz,进入到解压目录下执行:./bin/node -v来查看版本号，如果出现版本号说明安装正常。

 (3)设置软链接：

3.安装hexo

npm install hexo-cli -g
ln -s $path/nodejs/lib/node_modules/hexo-cli/bin/hexo /usr/local/bin/hexo

4.部署hexo

mkdir blog(创建自己的博客目录）
cd blog
hexo init 会自动把资源部署到该目录下

5.启动测试环境

执行hexo s 就可以预览你的博客了

在浏览器中输入localhost：4000就可以预览，取消预览可以执行ctrl + c

hexo进阶-将博客部署到github上

1.部署工具的下载：

npm install  -g cnpm  --registry=http://registry.npm.taobao.org
ln -s $path/nodejs/bin/cnmp  /usr/local/bin/cnmp
cnmp install --save hexo-deployer-git（推送工具）

2.github上博客仓库的创建

创建仓库名为:xxx.github.io(xxx为你的github用户名）

3.hexo的配置文件_config.yml修改

修改你博客目录下的_config.yml：在末尾修改
deploy:
  type: git
  repo: https://github.com/wangzhitao1997/wangzhitao1997.github.io.git
    branch: master

4.创建博客

hexo的博客默认是用markdown格式写的，在你的博客目录下执行 hexo new xxx（xxx为你的博客名）,xxx.md文件便会被创建在blog/source/_posts目录下
也可以将其他地方的md文件放到该目录下。

5.将你的博客上传到你的github的仓库中

hexo clean
hexo d (上传，输入你的github用户名以及密码就可以上传成功）

6.通过网页访问你的博客

浏览器输入https://wangzhitao1997.github.io/便可以访问你的博客了
