git的使用-将本地项目传到github仓库

发表于2020-11-20|更新于2020-11-20|github
|阅读量:
环境：window
1.进入项目的目录中，执行 git init(把文件夹变为git可以管理的仓库)

pc@LAPTOP-TJR9FCRR MINGW64 ~/Desktop/rsyslog-transmit-1.0 (master)
$ git add .
warning: LF will be replaced by CRLF in desktop/RsyslogTransmit-client.desktop.
The file will have its original line endings in your working directory

pc@LAPTOP-TJR9FCRR MINGW64 ~/Desktop/rsyslog-transmit-1.0 (master)
$ git commit -m “rsyslog-transmit codes first commit 2020.11.11”
[master (root-commit) 6226c49] rsyslog-transmit codes first commit 2020.11.11

pc@LAPTOP-TJR9FCRR MINGW64 ~/Desktop/rsyslog-transmit-1.0 (master)
$ git remote add origin https://github.com/wangzhitao1997/rsyslog-transmit.git

pc@LAPTOP-TJR9FCRR MINGW64 ~/Desktop/rsyslog-transmit-1.0 (master)
$ git push -u origin master (如果远程仓库是空的，需要加-u,下次再上传时不需要加-u)
